# pro-bridge-bot

Инструмент для обмена токенов в Web3 экосистеме.

## Особенности
- Поддержка Ethereum, Polygon, BSC
- Интеграция с популярными DEX
- Управление газом и проскальзыванием
- Безопасная работа с приватными ключами

## Установка
```bash
npm install
```

## Настройка
1. Создайте файл `.env` с вашими настройками:
```
PRIVATE_KEY=your-private-key
ALCHEMY_API_KEY=your-alchemy-key
```

## Использование
```bash
npm start
```

## Функции
- Проверка баланса ETH и ERC-20 токенов
- Обмен токенов через DEX
- Расчет оптимального газа
- Мониторинг транзакций

⚠️ **Внимание**: Это демонстрационный проект. Для продакшена добавьте дополнительную безопасность.
